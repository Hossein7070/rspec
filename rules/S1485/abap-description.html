Noncompliant Code Example

DATA BEGIN OF itab OCCURS n. "Noncompliant
  ...
DATA END OF itab [VALID BETWEEN intlim1 AND intlim2].

Compliant Solution

DATA BEGIN OF wa.
       ...
DATA END OF wa.
DATA itab LIKE TABLE OF wa.
