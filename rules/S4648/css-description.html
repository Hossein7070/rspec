This rule raises an issue when <code>font</code> or <code>font-family</code> properties contain a duplicated font name. This rule ignores <code>$sass</code>, <code>@less</code>, and <code>var(--custom-property)</code> variable syntaxes.

Noncompliant Code Example

a {
  font-family: 'Georgia', Georgia, serif; /* Noncompliant; 'Georgia' is duplicated */
}

Compliant Solution

a {
  font-family: Georgia, serif;
}
