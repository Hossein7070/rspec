== Exceptions

Use of comma operator is tolerated:
* in initialization and increment expressions of <code>for</code> loops.
----
for(i = 0, j = 5; i < 6; i++, j++) { ... }
----

* If the expression sequence is explicitly wrapped in parentheses.
