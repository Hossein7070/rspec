Conditional expressions which are always <code>true</code> or <code>false</code> can lead to dead code. Such code is always buggy and should never be used in production.

Noncompliant Code Example

SET FOO TO FALSE

IF FOO IS TRUE   *> Noncompliant
  DISPLAY "..."           *> never executed
END-IF.

IF FOO IS NOT TRUE OR BAR IS TRUE *> Noncompliant; BAR is never evaluated
  DISPLAY "..."
END-IF.

See

  • MITRE, CWE-570 - Expression is Always False
  • MITRE, CWE-571 - Expression is Always True
  • CERT, MSC12-C. - Detect and remove code that has no effect or is never executed
