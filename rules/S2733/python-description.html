The <code>exit</code> method is invoked with four arguments: self, type, value and traceback. Leave one of these out of the method declaration and the result will be a <code>TypeError</code> at runtime.

Noncompliant Code Example

class MyClass:
   def __enter__(self):
       pass
   def __exit__(self, exc_type, exc_val):  # Noncompliant
       pass
