The <code>await</code> keyword can only be used on "Awaitable" objects. Python has three types of awaitables: Future, Task and Coroutines. Calling await on any other object will raise a <code>TypeError</code>.

Noncompliant Code Example

import asyncio

def myfunction():
    print("myfunction")

async def otherfunction():
    await myfunction()  # Noncompliant. myfunction is not marked as "async"

asyncio.run(otherfunction())

Compliant Solution

import asyncio

async def myfunction():
    print("myfunction")

async def otherfunction():
    await myfunction()

asyncio.run(otherfunction())
