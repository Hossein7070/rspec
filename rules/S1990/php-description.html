There is no need to use the <code>final</code> modifier inside a <code>final</code> class. Everything in it is <code>final</code> by default.

Noncompliant Code Example

final class MyClass {

  public final String getName() {  // Noncompliant
    return name;
  }
}

Compliant Solution

final class MyClass {

  public String getName() {  // Compliant
    return name;
  }
}
